## Project Overview
This project demonstrates the deployment of a fully functional **Active Directory (AD)** environment from scratch using **GUI-based tools**. It covers:  
- Installing and configuring Domain Controllers  
- Creating Organizational Units (OUs)  
- Creating Users, Groups, and Computer Accounts  
- Configuring Group Memberships and Permissions  
- Documenting the entire setup process for professional reference  

**Objectives:**  
1. Deploy a secure AD domain environment  
2. Create structured OUs for better management  
3. Add users, groups, and computers  
4. Demonstrate GUI-based AD administration skills  
5. Provide clear documentation for reference
