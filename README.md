## Project Overview
This project demonstrates the deployment of a fully functional **Active Directory (AD)** environment from scratch. It covers:  
- Installing and configuring Domain Controllers (DCs)  
- Creating Organizational Units (OUs)  
- Creating Users, Groups, and Computer Accounts  
- Configuring group memberships and permissions  
- Documenting the setup process for hands-on learning and professional portfolio use  

**Objectives:**  
1. Deploy a secure AD domain environment  
2. Create structured OUs for better management  
3. Add users, groups, and computers  
4. Demonstrate PowerShell automation  
5. Provide clear documentation for reference
